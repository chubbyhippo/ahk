## Usage

```
Use Capslock to change between languages (Shift + Alt)
Use Shift + Capslock to Capslock
```
```
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/chubbyhippo/ahk/master/CapsToChangeInputLanguage.ahk" -OutFile "$Home\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\CapsToChangeInputLanguage.ahk"
```
