# MyAhk
```
Require AutoHotkey 
```
## For Windows 10

```
1. Compile CapsToChangeInputLanguage.ahk to CapsToChangeInputLanguage.exe
2. Set Run this program an administrator at the Compatibility tab in the Properties of CapsToChangeInputLanguage.exe
3. Create a shortcut to the batch file.
4. Once the shortcut has been created, right-click the file and select Cut.
5. Press the Start button and type Run and press enter.
6. In the Run window, type shell:startup to open the Startup folder.
7. Once the Startup folder has been opened, click the Home tab at the top of the folder and select Paste to paste the shortcut into the folder.
```

## Usage

```
Use Capslock to change between languages (Shift + Alt)
Use Shift + Capslock to Capslock
```
